🌊 Raising Water

AI Powered Flood Prediction System

Predict flood occurrence using Artificial Intelligence and Machine Learning based on weather conditions and rainfall parameters. The system provides quick and reliable predictions to support better decision making.

About the Project

Raising Water is an AI-powered Flood Prediction System developed using Python, Flask and Machine Learning. It predicts the possibility of floods by analyzing temperature, humidity, cloud cover and seasonal rainfall data using trained machine learning models.

🤖 AI Models

Uses Decision Tree, Random Forest, K-Nearest Neighbors and XGBoost algorithms for flood prediction.

⚡ Fast Prediction

Generates predictions instantly using the trained machine learning model and saved scaler.

🌦 Weather Analysis

Analyses weather parameters, rainfall statistics and environmental conditions before making predictions.

📊 High Accuracy

Multiple machine learning algorithms were evaluated and the best performing model is used for deployment.
